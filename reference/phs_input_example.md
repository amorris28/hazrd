# PHS Input Example

A set of random genotype data

## Usage

``` r
phs_input_example
```

## Format

### `phs_input_example`

A data frame with 4,000 rows and 1,996 columns:

- age:

  Age of diagnosis or age of censoring for each subject

- status:

  Case/control status

- c11_pos1707163, ...:

  Genotype at each locus
